Gadget Timeline

Gadget Timeline is a project loosely inspired on gdgt, meant to display your shiny gadgets on a timeline manner.

Please feel free to fork this project and create your own timeline.

How to edit the timeline

All you have to do is fork this project and edit the mygadgets.js file.

It has two arrays, categories and gadgets:

var categories = [
  {id: 1, content: 'Sample Category'},
  {id: 2, content: 'Yet another gadget category'}
];

var gadgets = [
  {
    group: 1,
    start: '2014-01-01',
    end: '2015-06-30',
    brand: 'Brand',
    device: 'Device Name',
    uri: 'https://en.wikipedia.org/wiki/Lorem_ipsum',
    photo_uri: 'http://lorempixel.com/image_output/technics-q-c-640-480-10.jpg'
  },
  {
    group: 2,
    start: '2015-07-01',
    brand: 'ACME Corp.',
    device: 'Shiny Gadget',
    wikipedia: 'https://en.wikipedia.org/wiki/Lorem_ipsum',
    photo_uri: 'http://lorempixel.com/image_output/technics-q-c-640-480-1.jpg'
  }
];

Each category has a numeric id and a content name. Each gadget has a group id (refering the category), a start date (on YYYY-MM-DD format), an optional end date, a brand name, a device name, and could have a uri for product page, a wikipedia uri and a photo_uri. Simple like that. :-)
